                    # Service de chat avec Java Socket #

Ce projet implémente un service de chat en utilisant les sockets JAVA.


** Déploiement et test **

   1. Créer un nouveau projet 
   2. Copiez les fichiers source (ChatServer.java, Client.java) dans le dossier de votre projet.
   3. S'assurer que le port 12345 n'est pas occupé
   3. Lancer le ChatServeur. Un message doit etre afficher ('Le serveur de chat est en ligne sur le port 12345')
   4. Lancer le Client et entrer le nom d'utilisateur. Un message doit etre diffusé (nom_d'utilisateur a rejoint le salon de discussion).répéter cette étape le nombre de fois voulu puis ecrire les messages voulu diffuser pour chaque client.Prsser Entrer dans les autres clients pour voir les messages envoyes
