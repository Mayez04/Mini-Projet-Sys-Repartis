                           # Gestion de liste de tâches avec Java RMI #

Ce projet implémente un système de gestion de liste de tâches en utilisant Java RMI.


** Déploiement et test **

   1. Créer un nouveau projet 
   2. Copiez les fichiers source (Server.java, RmiTraitImpl.java, RmiTrait.java, Client.java) dans le dossier de votre projet.
   3. S'assurer que le port 1125 n'est pas occupé.Sinon changer le dans le code du serveur
   3. Lancer le serveur. Un message doit etre afficher ('Server RMI started on port 1125')
   4. Tester la fonction d'ajout d'une tache en enlevant les commentaires indiqués dans le code et compiler
   5. Tester la fonction de supreession d'une tache en enlevant les commentaires indiqués dans le code et compiler
